# P1-1: ISA 规范合约提取

## 接口规范

- 输入：wiki 规范文档（见下方清单）
- 输出：`contracts/isa/spec.md`
- 约束：基于 SimRISC 0.5.3，每条断言标注来源

## 输入文件清单

| 文件 | 用途 |
|------|------|
| `SimRISC-00-指令系统设计.md` | 指令编码设计、QFC 编码表 |
| `SimRISC-01-数据类指令.md` | RD 寄存器组指令语义 |
| `SimRISC-02-地址类指令.md` | RB/RA 寄存器组指令、控制流 |
| `SimRISC-03-浮点类指令.md` | RF 寄存器组指令 |
| `SimRISC-04-系统类指令.md` | 系统指令 |
| `DADAO-11-AEE-应用程序运行环境.md` | 寄存器模型 |

## 验收标准

1. 版本号为 SimRISC 0.5.3
2. 包含以下章节：
   - §1 寄存器模型
   - §2 指令编码
   - §3 标量整数指令
   - §4 地址/内存指令
   - §5 控制流
   - §6 浮点指令
   - §7 系统指令
   - §8 NOP 与保留编码
   - §9 异常总结
   - 附录 A：完整编码清单
   - 附录 B：条件标志参考
3. 每条规范性断言标注来源 wiki 章节（如 `[SimRISC-01 §3.5]`）
